# fitness-data-visualization-project
Fitness Analytics Dashboard built with Power BI to track gym performance, members demographics, calorie calculations, and financial metrics. Includes BMI calculator, membership segmentation, and interactive visuals for actionable insights.


## Overview
Interactive dashboard to track fitness center performance, members demographics, calorie calculations, and financial metrics.  
Built using **Power BI** with integrated data sources.

## Key Features
- Members by age & gender.
- Members by age & status (Active vs Expired).
- BMI & Calorie Calculator (BMR, TDEE, Maintenance, Weight Loss).
- Financial performance (Revenue, Expenses, Profit).
- Membership segmentation (Platinum, Gold, Silver).
- Member progress tracking (% completion).

## Data Sources
- Members dataset (Excel/CSV).
- Trainer & financial records.

## Tools Used
- **Power BI** (Dashboard design).
- **Power Query** (Data cleaning).
- **DAX** (Custom measures for BMI, Calories, Progress).

## How to Run
1. Download the `.pbix` file.
2. Open with Power BI Desktop.
3. Connect to the provided dataset in `/data`.
4. Explore interactive visuals.


## Conclusion
This dashboard provides actionable insights for fitness centers to monitor members, health metrics, and financial performance.
